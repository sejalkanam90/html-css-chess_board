# ♟️ Chess Board UI

## Project Details

**Project Name:** Chess Board UI

**Language:** HTML & CSS

**Type:** Static Webpage

**Purpose:** To display a complete 8×8 chessboard with all pieces in their starting positions.

---

## About This Project

This project shows a standard chessboard with:

- **64 squares** (32 black, 32 white)
- **32 chess pieces** placed correctly
- Black pieces on top (rows 1-2)
- White pieces at bottom (rows 7-8)
- Empty middle rows (3-6)

---

## How It Works

The chessboard is made using:

1. **HTML div elements** - Each square is a separate box
2. **CSS styling** - Alternating black/white colors
3. **Unicode symbols** - For chess pieces (♔♕♖♗♘♙♚♛♜♝♞♟)
